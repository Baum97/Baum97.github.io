# Sebastian Baum – Portfolio

Personal portfolio built with Angular 17, deployed to [baum97.github.io](https://baum97.github.io).

## Development server

Run `npm start` and navigate to `http://localhost:4200/`.

## Build

Run `npm run build` – production artifacts land in `dist/baum97-portfolio/`.

## Tests

Run `npm test` to execute unit tests via Karma.
